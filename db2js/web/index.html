db2js

db2js 是一套前后端数据访问框架，前端设计与 ADO.net的DataSet近似，但不采取灵敏的数据绑定方案，而是通过主动调用 db2js.render和db2js.collect手工绑定和收集变动。
db2js后端使用 jdk8 上的 js 引擎 nashorn 实现了一套数据接口，设计理念为让一个文件支持多种操作，支持 restful 方式访问。
技术要点

jssp

JSSP是JavaScript Server Page的简称，采用js语言编写服务器端脚本，与jsp类似。不同之处主要有：
1. JSSP用[% %] 作为代码块括号
2. JSSP有[%= %]，[%~ %] [%- %] 等三种输出方式
3. JSSP 是DB2JS的变种。
JSSP页面转换后生成的代码为 
	db2js.jssp = function(params){
		out.print(‘\n’);
		out.print(‘\n’);
	...
	}
因此，jssp也可以使用所有db2js技术。
